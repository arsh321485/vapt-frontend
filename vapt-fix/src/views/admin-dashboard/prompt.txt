Frontend Developer ko yeh batao
Problem: Jab URL mein uid aur token hote hain (email link), Vue component tabs hide kar deta hai aur sirf form dikhata hai.

Fix — Auth.vue mein yeh 2 cheezein karni hain:

1. Tabs hamesha show karo

<!-- GALAT: tabs tab hide ho jate hain jab mode=set-password ho -->
<div v-if="!isSetPasswordMode" class="tabs">...</div>

<!-- SAHI: tabs hamesha dikhao -->
<div class="tabs">
  <button :class="{ active: activeTab === 'set-password' }">Set Password</button>
  <button :class="{ active: activeTab === 'signin' }">Sign In</button>
</div>
2. URL params se tab select karo

// mounted() ya onMounted() mein
const params = new URLSearchParams(window.location.search);
const mode = params.get('mode');      // 'set-password'
const uid = params.get('uid');         // user id
const token = params.get('token');     // reset token

if (mode === 'set-password') {
  this.activeTab = 'set-password';    // Set Password tab active
  this.uid = uid;
  this.token = token;
}
